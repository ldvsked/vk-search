# VK Search

Учебный проект для практики VK Education.

Кейс 2: поисковик по информационному ресурсу с парсингом данных, поисковым индексом и LLM-суммаризацией.

## MVP

- Парсинг информационного ресурса.
- Загрузка данных в поисковый индекс.
- Сервер, принимающий пользовательский запрос и возвращающий поисковую выдачу.
- Суммаризация результата с помощью LLM.

## Стек

- Python — парсинг, очистка, загрузка и индексация данных.
- PostgreSQL — хранение источников, документов, чанков, пользователей и логов.
- OpenSearch — полнотекстовый поиск BM25.
- Go — backend API.
- LLM API — суммаризация найденных документов.
- Docker Compose — запуск инфраструктуры.
